# IT Professional Skills Project - Gaming Accounts Reminder (GAR)

## A functional code written in C language, that is supposed to be a space for your many accounts in specific games, game related platforms (like Steam or Epic Games) or anything of that nature.


This code includes 2 additional libraries that are not by default used in C, which are stdlib.h and string.h and both are necessary for running this program,
since one is responsible for String elements working and the other for memory allocation. You should preferably use an IDE like virtual studio code to run this application (I created it in Virtual Studio 2022 so I would recommend that one to use the code). The code includes a few functions which I have learned recently in my classes, which include memory allocation, meaning this code may not be the best code ever, however it works as intended so that should be enough to fulfil its purpose. Here are the setup instructions:

1. download an IDE like Virtual Studio 2022.
2. copy or clone the code from this repository on your machine, debugg it and then run the code.
3. follow the instructions set in the menu and inside the code
4. Additional note: when allocating memory for the accounts (first interaction with the code i. e. **picture below**) give yourself extra space so that you can work without an issue.
